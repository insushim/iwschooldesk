# SchoolDesk

선생님을 위한 데스크톱 위젯 앱 — 시간표·할일·체크리스트·학급 체크·학생 기록 등을 바탕화면 위젯과 대시보드에서 한 번에.

바이브 코딩으로 직접 제작 · Electron + React + TypeScript + better-sqlite3.

## 설치

Windows: [Releases](../../releases) 탭에서 최신 exe 다운로드
- `SchoolDesk-Setup-1.0.0.exe` — 설치형 (권장)
- `SchoolDesk-Portable-1.0.0.exe` — 포터블 (USB 등)

## 주요 기능

- 시계·달력·할일·메모·시간표·체크리스트·D-Day·루틴·우리반 목표
- **학생용 시간표 위젯** — 전자칠판용 대형 표시. 수업 시작/끝 종소리 + 시계 위젯에 시각 알림
- **학급 체크 위젯** — 우유·양치·건강관찰 등 학생이 직접 체크, 자정 초기화
- **학생 기록 위젯** — 비밀번호 잠금 · 수정 이력 전부 해시체인으로 기록 (법원 증거 목적 가능)
- **학사일정 파일 자동 취입** — HWP · XLSX · DOCX · CSV · ICS 지원
- **바탕화면 위젯 모드** — 클릭 통과 + 맨 뒤 고정. 수업 중 방해 없이 배경처럼 보여줌

## 데이터 저장 위치

모든 데이터는 **로컬**에만 저장됩니다. 외부 서버 통신 없음.

- Windows: `%APPDATA%\SchoolDesk\schooldesk.db`

## 🔒 보안 — 선생님이 꼭 알아야 할 점

### 개인 PC에서 쓰실 때

안심하고 사용하셔도 됩니다.
- 네트워크 통신이 전혀 없어 외부 해킹/유출 경로가 없음
- 학생 기록 위젯은 scrypt 비밀번호로 앱 UI 열람을 차단
- 모든 수정이 SHA-256 해시체인 로그로 append-only 기록

### 공용 PC (교무실 공용 계정·교실 멀티유저)는 자제해 주세요

- SQLite DB(`schooldesk.db`)는 **암호화되어 있지 않습니다**. 같은 계정으로 로그인할 수 있는 사람은 DB 파일을 직접 열어 내용을 읽을 수 있어요.
- 교실 PC처럼 **Windows 로그인 비밀번호로 잠겨 있는 개인 계정**에서 사용하시면 실무상 안전합니다.
- 공용 PC에서 꼭 써야 한다면:
    1. 개인 Windows 계정으로만 사용
    2. 자리 비울 땐 Windows 화면 잠금 (`Win + L`)
    3. 학생 기록 위젯은 한 번 잠금 해제해도 **10분 유휴 시 자동 잠금**되지만, 선제적으로 🔒 버튼을 눌러 잠그는 습관 권장

### 법원 증거로 쓰려면

- 학생 기록 위젯 헤더의 ⬇ **"로그 내보내기"** 로 JSON 파일을 별도 매체(이메일·구글드라이브)에 **즉시 백업**해 두세요.
- 해시체인은 변조를 **탐지**할 수 있을 뿐, 원본 데이터 자체를 지켜 주진 않습니다. 백업본이 결정적입니다.

## Privacy (개인정보 처리)

SchoolDesk 는 **학생·교사 개인정보를 개발사 서버로 전송하지 않습니다.** 텔레메트리·자동 동기화·클라우드 백업·광고 추적 일체 없음. 모든 데이터는 사용자 PC 의 `%APPDATA%\school-desk\` (Windows) 에 로컬 SQLite 로 저장되며, 학생 기록·메모는 AES-256-GCM 봉투 암호화 + scrypt 키 도출 + 해시체인 변조 탐지로 보호됩니다.

외부 호출은 NEIS(급식·학교 검색), 기상청(날씨), 에어코리아(미세먼지) 만 사용하고, **학교 이름·좌표·도시명만 전송**합니다 — 학생 정보 미포함.

| 항목 | 설명 |
|---|---|
| 개인정보처리자 | 사용 교사·학교 (개인정보보호법 §2.5). 개발사는 처리자 아님 |
| 데이터 위치 | 사용자 PC 로컬 SQLite (`%APPDATA%\school-desk\`) |
| 암호화 | AES-256-GCM + scrypt + safeStorage(OS DPAPI) |
| 제3자 공유 | 없음 |
| 권장 운영 | OS 잠금·BitLocker·백업 USB 별도 보관·학년 종료 시 삭제 |

자세한 내용은 앱 안 **설정 → 개인정보** 탭 참조.

| **NEIS 교육행정정보시스템** ([open.neis.go.kr](https://open.neis.go.kr/)) | 급식 위젯 학교 정보·식단 | 교육부 open API |
| **Open-Meteo** ([open-meteo.com](https://open-meteo.com/)) | 기상청 장애 시 자동 백업 | CC BY 4.0 |
| **OpenTimestamps** ([opentimestamps.org](https://opentimestamps.org/)) | 학생기록 해시체인 Bitcoin 블록체인 등록 | 오픈소스 |
